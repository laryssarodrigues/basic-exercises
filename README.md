# Java Practice Exercises
This repository contains a collection of independent exercises developed during the **Programming Laboratory (2026.1)** course.  
Each `.java` file in this project corresponds to a specific question from the exercise list, focused on consolidating programming logic and Java syntax.

---
## |Technologies & Tools|
* **Language:** Java
* **JDK:** 24 (OpenJDK)
* **IDE:** Intellij IDEA

## |How to run|
Since the classes are independent, you can run each question individually:
1. Open the project folder in your IDE;
2. Locate the file for the question you want to view;
3. Run the `main` method prresent in each class.
